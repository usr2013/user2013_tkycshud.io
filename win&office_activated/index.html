    Microsoft Activation Scripts (MAS)

    An open-source Windows and Office activator featuring the following activation methods: - HWID - Ohook - TSforge - KMS38 - Online KMS

    Additionally, it includes advanced troubleshooting features.

    Download / How to use?

    Method 1 - PowerShell (Windows 8 and later) ❤️

    1. Open PowerShell (not CMD). To do this:
    2. Right-click on the Windows start menu.
    3. Select PowerShell or Terminal.
    4. Copy and paste the code below into PowerShell and press Enter: powershell irm https://get.activated.win | iex Alternatively, you can use the following (this will be deprecated in the future): powershell irm https://massgrave.dev/get | iex
    5. You will see the activation options:
    6. Choose (1) HWID for Windows activation.
    7. Choose (2) Ohook for Office activation.

    That's all!

    Troubleshooting ❓

    • If you are unable to launch MAS using the PowerShell method, please refer to Method 2 above.
    • If MAS is launched and the script shows any errors:
    • Check for any troubleshooting steps mentioned in blue color.
    • Try to follow those steps.

    If you have any issues, feel free to reach out to us here.

    Note

    • The IRM command in PowerShell downloads a script from a specified URL, and the IEX command executes it.
    • Always double-check the URL before executing the command and verify the source if manually downloading files.
    • Be cautious, as some spread malware disguised as MAS by using different URLs in the IRM command.
